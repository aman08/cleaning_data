The dataset includes the following files:
=========================================

- README.md
- Codebook.md : describes the variables in the dataset and steps involved in data transformation
- run_analysis.R : R script to create a tidy dataset

Notes: 
======
- Download the source raw file from the following link
https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
- Unzip it under a folder called R in the C directory
- Set folder to R - setwd("C:/R")
- Run the script from the working directory
	source('run_analysis.R')
- Script will create a tidydata.txt in the working directory
